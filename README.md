# Customer Churn Analysis at Fit.ly

## Overview

This project analyzes customer churn for **Fit.ly**, a subscription-based fitness application.

The objective was to identify factors associated with customer churn by combining customer account information, customer support data, and user activity.

---

## Tools Used

- Python
- Pandas
- Matplotlib
- Jupyter Notebook
- Git
- GitHub

---

## Project Workflow

- Data Validation
- Data Cleaning
- Exploratory Data Analysis
- Data Integration
- Customer Churn Analysis
- Business KPI Definition
- Business Recommendations

---

## Key Findings

- Overall churn rate: **28.5%**
- The Free subscription plan had the highest churn rate.
- Customers with longer support resolution times were more likely to churn.
- Lower user engagement was strongly associated with churn.
